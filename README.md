# Pomodoro Timer

macOS向けのデスクトップウィジェット風ポモドーロタイマーアプリです。

![macOS](https://img.shields.io/badge/macOS-14.0+-blue)
![Swift](https://img.shields.io/badge/Swift-5-orange)

## 特徴

- **ウィジェット風UI** - フロストガラス(半透明)デザインのコンパクトなウィンドウ
- **他のウィンドウの後ろに配置** - 作業の邪魔にならず、デスクトップを表示すると確認できる
- **ドラッグで自由に配置** - 好きな位置に移動可能
- **3つのモード** - 集中 / 小休憩 / 長休憩
- **2つのプリセット** - スタンダード(25分) / ディープフォーカス(50分)
- **時間のカスタマイズ** - +/- ボタンで1分単位の調整が可能
- **メニューバー表示** - タイマー実行中は残り時間をメニューバーに表示
- **統計機能** - 今日のセッション数、集中時間、連続日数、週間チャート
- **完了通知** - セッション完了時にサウンドで通知
- **フローティングオーバーレイ** - ウィンドウを閉じてもタイマー実行中は小窓で残り時間を表示

## 必要環境

- macOS 14.0 (Sonoma) 以降

## ビルド・実行

```bash
swift build
.build/debug/PomodoroTimer
```

## 使い方

1. アプリを起動するとデスクトップ右下にウィジェットが表示されます
2. ▶ ボタンでタイマー開始、⏸ で一時停止
3. 停止中に +/- ボタンで時間を調整
4. 下部のタブで集中 / 小休憩 / 長休憩を切り替え
5. ページドットで統計ページに切り替え
6. メニューバーのタイマーアイコンでウィンドウの表示/非表示を切り替え
